Welcome to Millionare - The game


We have created a game based on the popular TV show called "Who wants to be a millionare?"

The game is based on a quiz. Each correct answer makes you go up one step in the prizelist.

If you answer wrong, You lose! There are a checkpoint for every fifth correct answer.

If you reach one of the checkpoints, you are guarenteed that prize.

There are four lifelines alivable. "Phone a friend", "Ask the audience", "50/50" and "Change Question".
